Tarea Android - Gestor de Tareas

Esta es una aplicación Android desarrollada en **Java** con **Android Studio**. Permite al usuario agregar, editar, eliminar y buscar tareas.

---

Funcionalidades

-  Agregar nuevas tareas  
-  Editar tareas existentes  
-  Eliminar tareas  
-  Buscar tareas por nombre  
-  Almacenamiento persistente con `SharedPreferences`



 Requisitos

- Android Studio (**recomendado: Electric Eel o superior**)
- Emulador o dispositivo físico con **API 30+**



 Cómo ejecutar

1. Clona este repositorio  
   `git clone https://github.com/RichardMP7/Tarea_CompMovil.git`
2. Abre el proyecto en Android Studio  
3. Ejecuta la app en un emulador o dispositivo Android físico



 Notas

- La persistencia se maneja con `SharedPreferences`, por lo que las tareas se conservan entre sesiones.
- Este proyecto es parte del taller de clase de Computación Móvil utilizando Android Studio.
